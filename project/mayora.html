Project Detail - Mayora Monitoring

Dashboard Monitoring - Real-Time Production Data

Dashboard Monitoring adalah sistem yang menampilkan data produk secara real-time selama proses produksi berlangsung. Dashboard ini menyediakan informasi penting seperti:

  • Qty Actual: Jumlah aktual produk yang telah diproduksi.
  • No Good (NG): Jumlah produk yang tidak memenuhi standar kualitas.
  • Qty Planning: Jumlah produk yang direncanakan untuk diproduksi.

Fitur utama pada dashboard ini adalah Report Product, yang memungkinkan pengguna untuk mengunduh hasil produksi dalam berbagai format file, termasuk:

  • CSV
  • Excel
  • PDF

Role User:

  • Supervisor: Memiliki akses untuk mengunduh laporan hasil produksi dalam format Excel dan PDF.
  • Admin: Memiliki akses penuh untuk mengunduh laporan dalam semua format yang tersedia (CSV, Excel, dan PDF).

Sistem ini membantu memonitor dan mendokumentasikan hasil produksi secara efisien, memastikan data dapat diakses oleh pengguna sesuai dengan peran dan hak akses masing-masing.
