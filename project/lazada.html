Project Detail - Lazada Sorting

Scada Server - Sorting Data Management

Scada server adalah sistem informasi yang menerima data dari Scada client. Scada client bertanggung jawab untuk proses sorting barang, di mana setiap data sorting akan diteruskan ke Scada server untuk diproses lebih lanjut.

Fitur Cute/Sortir: Sistem ini memiliki 14 sortir. Setiap parcel yang masuk akan terdeteksi dan terhitung secara otomatis. Informasi sorting ini memudahkan pengelolaan dan pelacakan parcel berdasarkan kategori sortirnya.

Fitur Read Rate: Data read rate diperoleh dari vendor kamera. Kamera akan membaca setiap parcel yang melewati jalur sorting. Informasi ini menunjukkan apakah parcel berhasil terbaca oleh kamera atau tidak, sehingga dapat meningkatkan akurasi dalam proses tracking.

Fitur Parcel Size: Informasi dimensi parcel (lebar, tinggi, dan berat) diperoleh dari vendor checkweigher. Setiap parcel yang masuk akan diukur secara otomatis, sehingga data ini dapat digunakan untuk pengelompokan dan analisis ukuran barang.

Sistem ini memberikan informasi terintegrasi yang mendukung efisiensi dan akurasi dalam pengelolaan data sorting parcel secara real-time.
